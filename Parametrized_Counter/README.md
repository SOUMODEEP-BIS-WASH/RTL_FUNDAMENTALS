
# Parameterized Counters

## Overview

This project implements parameterized up-counter and down-counter modules in Verilog HDL.

Both counters support reset, preset loading, and enable control for flexible counting operations.

## Features

* Parameterized bit width
* Up-counter implementation
* Down-counter implementation
* Reset support
* Preset loading
* Enable control

## Verification

Verified for:

* 2-bit counters
* 4-bit counters
* 8-bit counters

Test cases covered:

* Increment/decrement
* Preset load
* Reset
* Hold state

## Files

* `up_counter.v`
* `down_counter.v`
* `tb.v`

## Tools Used

* Verilog HDL
* Icarus Verilog
* GTKWave
