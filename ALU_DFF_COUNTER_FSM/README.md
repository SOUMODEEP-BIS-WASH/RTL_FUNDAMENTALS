# ALU + Register + Counter + FSM Alert Mechanism

## Overview

This project implements an integrated digital system combining an ALU, register, counter, and FSM-based alert logic.

The system monitors ALU output changes and generates an alert after detecting a specific number of output transitions.

## Architecture Components

* ALU
* Register
* Counter
* FSM Alert Logic

## ALU Operations

* Addition
* Subtraction
* Multiplication
* AND
* OR
* NAND
* NOR

## Features

* Multi-module integration
* Change detection mechanism
* Counter-based tracking
* FSM-based alert generation
