# Linear Feedback Shift Register (LFSR)

## Overview

This project implements a 4-bit Linear Feedback Shift Register (LFSR) in Verilog HDL for pseudo-random sequence generation.

The design uses XOR-based feedback logic to generate deterministic pseudo-random sequences.

## Features

* 4-bit implementation
* XOR feedback logic
* Sequential shifting
* Reset support

## Applications

* Built-In Self-Test (BIST)
* Random sequence generation
* Pattern generation

## Verification

Test cases covered:

* Reset behavior
* Shift operation
* Feedback sequence generation



## Tools Used

* Verilog HDL
* Icarus Verilog
* GTKWave
