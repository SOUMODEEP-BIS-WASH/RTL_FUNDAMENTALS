# Binary to Gray Converter

## Overview

This project implements a parameterized Binary-to-Gray code converter using combinational logic.

Gray code is widely used in digital systems where minimizing bit transitions is important.

## Features

* Parameterized width
* Combinational logic
* XOR-based conversion
* Scalable design

## Conversion Logic

* MSB remains unchanged
* Remaining bits are generated using XOR logic

## Verification

Exhaustive verification performed for all 4-bit input combinations.


## Tools Used

* Verilog HDL
* Icarus Verilog
* GTKWave
