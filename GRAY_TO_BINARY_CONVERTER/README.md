
# Gray to Binary Converter

## Overview

This project implements a parameterized Gray-to-Binary converter using combinational logic.

The design efficiently converts Gray code values back to binary representation.

## Features

* Parameterized width
* Combinational logic
* XOR-based conversion
* Scalable implementation

## Conversion Logic

Binary output is derived using iterative XOR operations.

## Verification

Verified using complete 4-bit Gray code sequence.


## Tools Used

* Verilog HDL
* Icarus Verilog
* GTKWave
