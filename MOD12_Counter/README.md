# Mod-12 Counter

## Overview

This project implements a modulo-12 counter in Verilog HDL.

The counter increments sequentially and resets back to zero after reaching the maximum count.

## Features

* Mod-12 counting
* Enable control
* Reset functionality
* Cyclic counting sequence

## Applications

* Digital clocks
* Timers
* Frequency division

## Verification

Test cases covered:

* Counter increment
* Counter rollover
* Reset behavior

## Files

* `design.v`
* `tb.v`

## Tools Used

* Verilog HDL
* Icarus Verilog
* GTKWave
