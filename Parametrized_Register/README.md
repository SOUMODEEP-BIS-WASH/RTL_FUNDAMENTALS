
# Parameterized Register

## Overview

This project implements a reusable parameterized N-bit register in Verilog HDL. The design supports synchronous reset and controlled data loading through a load enable signal.

The module can be instantiated for different register widths, making it scalable and reusable in larger digital systems.

## Features

* Parameterized register width
* Synchronous reset
* Load enable control
* Scalable architecture
* Reusable RTL design

## Verification

The design was verified using testbenches for:

* 4-bit register
* 8-bit register
* 16-bit register

Test cases covered:

* Reset operation
* Data loading
* Hold state



## Tools Used

* Verilog HDL
* Icarus Verilog
* GTKWave
